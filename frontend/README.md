## Tecnologias

- **Framework**: Next.js 14.2
- **Styling**: Tailwind CSS
- **TypeScript**: Full type safety
- **API**: RESTful via NestJS backend

## Desenvolvimento

```bash
# Com Docker
docker-compose up myclinic-next

# Local
npm install
npm run dev
```

Acesse em: http://localhost:3000

## Responsividade

- Mobile-first approach
- Breakpoints:
  - Mobile: < 768px
  - Tablet: 768px - 1024px
  - Desktop: > 1024px
